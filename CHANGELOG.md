# 更新日志

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 格式，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2024-12-01

### 新增

- 家谱管理：创建、编辑、删除家谱，支持启用/禁用状态控制
- 家族成员管理：完整的成员 CRUD 操作，支持世代计算和父子关系维护
- 事件管理：记录成员的出生、逝世、迁徙等事件，支持公历/农历日期
- 地点管理：事件关联地点的增删改查
- 事件类型管理：自定义事件类型分类
- 相册管理：按家谱组织相册，支持照片上传、缩略图生成
- 照片管理：照片批量上传、按成员/相册分类查看
- GEDCOM 导入导出：支持 GEDCOM 5.5.1 格式的家谱数据交换
- Excel 导入：从 Excel 文件批量导入成员数据
- 统计报告：家谱成员性别、世代、年龄、职业、居住地分布统计
- 祖先报告：向上追溯指定世代的祖先链
- 后裔报告：向下展开指定世代的后裔树
- HTML 报告生成：将统计报告导出为格式化的 HTML 页面
- 数据库备份与恢复：SQL 格式的数据库备份和恢复功能
- 系统状态监控：数据库连接状态、表记录数等系统信息
- 管理员管理：管理员账号的增删改查，密码强度验证
- JWT 认证：基于 Bearer Token 的身份认证
- 全局异常处理：统一的异常捕获和错误响应
- 请求限流：防止 API 滥用
- 安全头设置：X-Content-Type-Options、X-Frame-Options 等安全头
- XSS 防护：输入清理和 XSS 检测
- 操作日志：关键操作的审计记录
- 分页查询：所有列表接口支持分页和关键词搜索
- 内存缓存：常用查询结果的缓存优化
- Docker 部署：docker-compose 编排的容器化部署方案

### 技术栈

- 后端：ASP.NET Core 10 + SqlSugar ORM + MySQL 8.0
- 前端：Blazor Server
- 认证：JWT Bearer Token + BCrypt 密码哈希
- 部署：Docker + docker-compose
