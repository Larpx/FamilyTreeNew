## 1. 系统简介

### 1.1 系统概述

家族族谱管理系统是一个基于Web的家谱信息管理平台，旨在帮助用户便捷地记录、管理和展示家族成员信息。系统支持多种家谱展示方式，包括树形视图、世系图、关系图等，让家族历史得以数字化保存和传承。

### 1.2 系统特点

- **多维度展示**：支持树形视图、世系图、关系图、时间线等多种展示方式
- **Excel导入**：支持通过Excel批量导入家族成员信息
- **相册管理**：支持家族相册和照片管理
- **安全可靠**：采用JWT认证、数据加密等安全措施
- **响应式设计**：支持PC端和移动端访问

### 1.3 系统要求

- 现代浏览器（Chrome 80+、Firefox 75+、Edge 80+、Safari 13+）
- 建议屏幕分辨率：1920×1080或更高
- 稳定的网络连接

## 3. 快速开始

### 3.1 登录系统

1. 打开浏览器，访问系统地址
2. 在登录页面输入用户名和密码
3. 点击"登录"按钮进入系统

**注意事项：**
- 用户名长度为3-50个字符
- 密码长度为6-100个字符
- 连续登录失败5次将锁定账户15分钟

### 3.2 首页导航

登录成功后，您将看到系统首页，包含以下区域：

- **顶部导航栏**：系统Logo、功能菜单、用户信息
- **侧边栏**：功能模块快捷入口
- **主内容区**：当前选中功能的操作界面
- **底部信息栏**：版权信息、技术支持联系方式

### 3.3 修改密码

1. 点击右上角用户头像
2. 选择"修改密码"
3. 输入原密码和新密码
4. 点击"确认"完成修改

**密码要求：**
- 最少8个字符
- 包含大写字母
- 包含小写字母
- 包含数字
- 包含特殊字符（如!@#$%^&*）

## 4. 功能模块详解

### 4.1 家谱管理

#### 4.1.1 创建家谱

1. 进入"家谱管理"模块
2. 点击"新建家谱"按钮
3. 填写家谱信息：
   - **家谱名称**：必填，最多200字符
   - **家谱简介**：选填，最多5000字符
   - **是否需要验证**：开启后访客需验证才能查看
   - **是否启用**：控制家谱的可见性
4. 点击"保存"完成创建

#### 4.1.2 编辑家谱

1. 在家谱列表中找到目标家谱
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"完成修改

#### 4.1.3 删除家谱

1. 在家谱列表中找到目标家谱
2. 点击"删除"按钮
3. 确认删除操作

**注意：** 删除家谱将同时删除该家谱下的所有成员信息，此操作不可恢复！

### 4.2 成员管理

#### 4.2.1 添加成员

1. 进入家谱详情页
2. 点击"添加成员"按钮
3. 填写成员信息：

| 字段 | 说明 | 必填 |
|------|------|------|
| 姓氏 | 成员姓氏 | 是 |
| 名字 | 成员名字 | 是 |
| 父成员 | 选择父成员，用于建立关系 | 否 |
| 字号别称 | 别名或字号 | 否 |
| 排行 | 在兄弟姐妹中的排行 | 否 |
| 字辈 | 家族字辈 | 否 |
| 生辰公历 | 公历出生日期 | 否 |
| 生辰农历 | 农历出生日期 | 否 |
| 居住地 | 现居住地址 | 否 |
| 职业 | 职业信息 | 否 |
| 个人信息 | 详细个人介绍 | 否 |
| 小注 | 备注信息 | 否 |
| 卒亡标识 | 是否已故 | 否 |
| 卒亡公历 | 公历去世日期 | 否 |
| 卒亡农历 | 农历去世日期 | 否 |
| 备注 | 其他备注 | 否 |

4. 点击"保存"完成添加

#### 4.2.2 编辑成员

1. 在成员列表中找到目标成员
2. 点击"编辑"按钮
3. 修改相关信息
4. 点击"保存"完成修改

#### 4.2.3 删除成员

1. 在成员列表中找到目标成员
2. 点击"删除"按钮
3. 确认删除操作

**注意：** 如果成员有子成员，需要先删除或移动子成员才能删除该成员。

#### 4.2.4 成员搜索

- 支持按姓名关键词搜索
- 支持按世代筛选
- 支持按父成员筛选

### 4.3 数据导入

#### 4.3.1 下载导入模板

1. 进入家谱管理页面
2. 点击"下载模板"按钮
3. 保存Excel模板文件

#### 4.3.2 填写模板数据

按照模板格式填写成员信息：

| 列名 | 说明 |
|------|------|
| 姓氏* | 必填 |
| 名字* | 必填 |
| 父成员姓名 | 用于建立关系，需与已有成员姓名匹配 |
| 字号别称 | 选填 |
| 排行 | 选填 |
| 字辈 | 选填 |
| 生辰公历 | 格式：YYYY-MM-DD |
| 生辰农历 | 选填 |
| 居住地 | 选填 |
| 职业 | 选填 |
| 个人信息 | 选填 |
| 小注 | 选填 |
| 是否已故 | 是/否 |
| 卒亡公历 | 格式：YYYY-MM-DD |
| 卒亡农历 | 选填 |
| 备注 | 选填 |

#### 4.3.3 导入数据

1. 进入目标家谱详情页
2. 点击"导入成员"按钮
3. 选择填写好的Excel文件
4. 点击"开始导入"
5. 查看导入结果

### 4.4 家谱展示

#### 4.4.1 树形视图

以树形结构展示家族成员关系，支持：
- 展开/折叠节点
- 缩放视图
- 搜索定位成员

#### 4.4.2 世系图

以世代为单位展示家族传承，清晰展示各代成员。

#### 4.4.3 关系图

以网络图形式展示成员间的关系，支持拖拽和缩放。

#### 4.4.4 时间线

按时间顺序展示家族重要事件。

### 4.5 相册管理

#### 4.5.1 创建相册

1. 进入"相册管理"模块
2. 点击"新建相册"
3. 填写相册名称和描述
4. 点击"保存"

#### 4.5.2 上传照片

1. 进入目标相册
2. 点击"上传照片"
3. 选择照片文件（支持jpg、png格式）
4. 等待上传完成

#### 4.5.3 管理照片

- 查看照片详情
- 编辑照片描述
- 删除照片
- 设置封面照片

### 4.6 系统设置

#### 4.6.1 数据库状态

查看数据库连接状态和存储使用情况。

#### 4.6.2 数据备份

1. 进入"系统设置"模块
2. 点击"数据备份"
3. 选择备份范围
4. 点击"开始备份"
5. 下载备份文件

## 5. 常见问题解答

### 5.1 登录相关

**Q: 忘记密码怎么办？**

A: 请联系系统管理员重置密码。

**Q: 账户被锁定怎么办？**

A: 账户会在15分钟后自动解锁，或联系管理员手动解锁。

**Q: Token过期了怎么办？**

A: 系统会自动提示重新登录，请重新输入用户名和密码。

### 5.2 家谱管理

**Q: 如何批量添加成员？**

A: 使用Excel导入功能，先下载模板，填写数据后上传即可。

**Q: 删除家谱后数据能恢复吗？**

A: 不能，删除操作不可逆，请谨慎操作。建议定期备份数据。

**Q: 成员世代是如何计算的？**

A: 系统自动根据父成员关系计算世代，始祖为第1代，每往下一代加1。

### 5.3 数据导入

**Q: 导入失败显示格式错误怎么办？**

A: 请检查Excel文件格式是否符合模板要求，特别注意日期格式应为YYYY-MM-DD。

**Q: 导入时部分数据失败怎么办？**

A: 系统会显示详细的错误信息，请根据提示修改后重新导入失败的数据。

**Q: 导入的成员如何建立关系？**

A: 在"父成员姓名"列填写已有成员的姓名，系统会自动建立关系。

### 5.4 系统使用

**Q: 支持哪些浏览器？**

A: 推荐使用Chrome、Firefox、Edge最新版本，Safari 13及以上版本。

**Q: 数据会自动保存吗？**

A: 系统会自动保存您的操作，但建议在编辑完成后点击保存按钮确认。

**Q: 如何导出家谱数据？**

A: 在家谱详情页点击"导出"按钮，可选择导出为Excel或PDF格式。
